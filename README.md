# Tradia.co.nz Landing Page

Static GitHub Pages landing page for the sale of `Tradia.co.nz`.

## Files

- `index.html` - page markup
- `styles.css` - responsive styling
- `assets/` - supplied Tradia logo, banners, and design imagery
- `CNAME` - custom domain configuration for GitHub Pages

Open `index.html` in a browser or publish this folder with GitHub Pages.
